the program and specifying actions taken according to the conditions.

if conditions: this condition is apply when we have to compare the value.

syntax::>     
        if condition:
            statements

else conditions: it is apply when the if is false then after else condition is apply 

syntax::>
        if condition:
            statements
        else:
            statements
